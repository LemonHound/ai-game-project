# Setup Guide

## Python Backend Server Setup

Note that you'll need to adjust the path defined in package.json, under `dev:python` if you're going to use this script
to boot your server.

### Option 1: Conda

```bash
conda env create -f environment.yml
conda activate aigamehub
```

### Option 2: PyCharm

- Run app.py

### Option 3: VSCode

- Run app.py
